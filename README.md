Neon Tic-Tac-Toe Game

A modern and visually immersive Tic-Tac-Toe web application built using HTML, CSS, and JavaScript.
This project features a futuristic neon UI, smooth animations, and interactive gameplay for an engaging user experience.

🚀 Live Preview

👉 https://akshaya0323.github.io/SCT_WD_03/

📌 Project Overview

This project is part of a Web Development Internship Task (SCT_WD_03).
It demonstrates the implementation of a classic game with a modern UI/UX approach, focusing on animations, responsiveness, and design aesthetics.

✨ Features
🎮 Game Features

   ✔️ Two-player mode
   ✔️ Turn-based gameplay
   ✔️ Win & draw detection
   ✔️ Restart functionality

   ✔️ Neon glow effects

💫 Animations

   ✔️ Floating particles background

🧠 How It Works
   - 3×3 grid using CSS Grid
   -Players take turns (X & O)
   -JavaScript handles:
      *Game state
      *Win conditions
      *Draw logic
   -Winning cells animate + modal appears
